# Task Request Template

## Task ID
task-xxx

## Request summary
Describe the requested change in one sentence.

## Target files
List the files that should be changed.
- path/to/file.tsx

## Expected result
Describe what should be true after the change.

## Verification
Describe how the change should be checked.
- lint
- build
- browser/manual check
- API check
- other

## Risk note
Describe what could break or what must be watched carefully.

## Output format
Choose one:
- before/after
- exact insertion point
- full file

## Scope guard
State what should NOT be changed.

## Notes for Codex
Add any implementation hints, constraints, or non-goals.
